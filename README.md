# За спичкой: Наследие Кузнеца 🎮

## Версия: 1.2.0 "Танцующий Стикман"

### Как играть:
1. Установи Python 3.6 или выше
2. Скачай файл matchstick_game.py
3. Запусти: python matchstick_game.py

### Управление:
- Вводи цифры 1-5 для выбора
- Нажимай Enter для продолжения
- Найди секрет (пункт 5 в первой главе)!

### Секреты:
🕺 Танцующий стикман
✨ Бонус +50 очков
🎭 Секретная концовка
